# Menu Driven App
C++ Simple Menu Driven app activity with validation

## Requirements
- Menu
- Validation
- Accept User Input (**integer**) and store it in Array.
- Display the Array.
- Find the **Sum** of the Elements in the Array.
- Find the **Average** of the Elements in the Array.
- Find the **Max** and **Min** Element in Array.
- Display the **Reverse** Elements in Array.
- **Sort** and Display the Elements in Array
- User can **Search** Element in the Array.
